# E-1402 log rotation failed

## Meaning

The system could not rotate the active log or create the next session segment.

## Common causes

- Session root is missing, read-only, or full.
- Generated path conflicts with an existing file or directory.
- Metadata or index files could not be finalized before rotation.

## Impact

Ingest may pause or fail for the affected source. Existing records remain in the
previous segment unless the accompanying message states otherwise.

## Operator action

- Check the session root path and permissions.
- Confirm there is enough disk space for the next segment.
- Inspect the current session directory for partial files.
- Restart the source after storage issues are resolved.

## Developer notes

Rotation names and on-disk layout are compatibility-sensitive. Update log-format
fixtures if behavior changes intentionally.
