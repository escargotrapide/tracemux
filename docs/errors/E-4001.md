# E-4001 UI wire frame ignored

## Meaning

The web UI received a WebSocket frame that could not be unpacked or did not
match the `tracemux.v1` envelope shape.

## Impact

The malformed frame is ignored. The connection stays open and future valid
frames can still be processed.

## User remedy

- Refresh the UI if live state looks stale.
- Confirm the browser is connected to a TraceMux server using the same wire
  protocol version.
- If the error repeats after a server restart, capture browser console output
  and server logs for debugging.

## Developer notes

Keep this error for browser-side decode/envelope failures. Server-side malformed
wire frames should use `E-2001` or `E-2002`.
