# E-1001 pipeline generic

## Meaning

A pipeline, export, import, or server-side operation failed without a more
specific public error code.

## Common causes

- Internal I/O failed while reading, writing, exporting, or bundling data.
- A lower-level component returned an unexpected error that has no narrower
  public code yet.
- A source, sink, framer, or decoder error crossed a boundary without a more
  specific mapping.

## Impact

The requested operation fails. Existing persisted sessions should remain intact
unless the accompanying message says a write, fsync, or rotation failed.

## Operator action

- Read the full error message after the code; it should include the failing
  operation or context.
- Retry once if the message points to a transient file or network operation.
- Check server logs around the same timestamp for the root cause.
- Escalate with the session id, source id, command, and log excerpt if it is
  repeatable.

## Developer notes

Use a narrower code when the failure belongs to a known layer. Keep `E-1001`
for true catch-all pipeline failures.
